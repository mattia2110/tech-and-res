#To develop this alternate history from 1945 to 2025, while maintaining the premise of a China that has not experienced the collapse of the Warlord era but remains anchored to its structural paralysis, we must draw on the demographic, economic, and institutional dynamics that emerged in the nineteenth century.
#In this scenario, the country continues to suffer from the massive demographic pressure and competition for resources that have historically threatened the stability of the state. However, the system of bureaucratic federalism (fengjian applied to the provincial system) and military decentralization prevent a violent fracture. Foreign powers, however transformed by their own historical developments, continue to maintain commercial interests in China, forcing Beijing to perpetuate its balancing diplomacy.
#Here are three possible evolutionary paths this China could pursue between 1945 and 2025:
#Branch 1: Corporate Confederation (The Economic Evolution of "Guandu Shangban")
# This branch explores the definitive fusion between provincial administration and commercial capital, taking the guandu shangban (government supervision and mercantile management) business model to its extreme.
# 1945-1980: With the progressive loosening of direct imperial control (or a nominally republican but weak central government), provincial governors and the comprador class of coastal ports forge an iron-clad alliance. The compradors, originally acting as intermediaries for foreign powers, use their immense capital to finance regional industrial conglomerates. The governors provide military protection and legal monopoly in exchange for profit shares. The internal tax system, based on the fragmentation of the likin tax, is modernized into actual inter-provincial customs barriers.
# 1980-2025: China transforms into a decentralized corporate oligarchy. A unified national bourgeoisie does not emerge, but rather a series of "provincial mega-corporations" competing with each other on the global market. Coastal cities become hyper-capitalist technological and financial hubs, governed by boardrooms where foreign powers still hold minority but influential stakes.
# Branch 2: The "United Provinces" (The Political Evolution of Federalism)
# This branch focuses on the peaceful institutionalization of decentralized reformist thought inspired by the philosopher Gu Yanwu, who theorized a centralized state system but with autonomous local governance.
# 1945-1990: Faced with growing social tensions and the threat of peasant revolts, provincial elites realize that the only way to avoid systemic collapse is to formalize their autonomy. The regional armies, which emerged in the nineteenth century to quell revolts, crystallize into provincial self-defense forces. To avoid mutual destruction, the provinces enter into a constitutional pact based on the ancient ideal of local self-government (difang zizhi), effectively creating the "United States of China."
# 1990-2025: A long period of internal "Cold War" begins. The coastal provinces, richer and more connected, and the inland provinces, more populous and agricultural, constantly balance each other. The Central Government manages foreign policy exclusively, applying the doctrine of "using barbarians against barbarians" to prevent external interference in inter-provincial conflicts. The ancient frontiers (Manchuria, Xinjiang) are definitively saturated by population flows controlled by the center to act as shields against the outside world.
# Branch 3: Stagnant Utopia and Techno-Confucianism (Socio-Cultural Evolution)
# This branch explores the triumph of the Zhongti Xiyong philosophy (Chinese knowledge as foundation, Western knowledge as practical application) combined with the utopian and cyclical visions of late-nineteenth-century Chinese proto-science fiction.
# 1945-2000: The Central Government and the rural gentry (Shenshi) conclude that mass industrialization and Western-style urbanization would inevitably lead to the destruction of Confucian values ​​and the collapse of civilization under the weight of class struggle. The state therefore decides to deliberately limit the industrial development of civil society. The adoption of foreign technology is limited exclusively to the development of social control infrastructures and the passive maintenance of borders.
# 2000-2025: The ancient institution of "Always Normal Granaries" (changpingcang), used to stabilize prices and prevent famine, is updated with advanced logistics and information networks imported from abroad. However, the population is kept in a state of pre-modern rural stasis. Literate officials are transformed into techno-bureaucrats who use technology to calculate agricultural cycles, prevent ecological disasters, and nip dissent in the bud, ensuring a minimal but universal subsistence.

the_future_of_china.7 = {
	type = country_event
	title = the_future_of_china.7.t
	desc = the_future_of_china.7.d

	}

	trigger = {
		var:ztr_china_corporate_var > 60
		any_country = {
			country_rank >= rank_value:great_power
			ztr_is_individualist = yes
		}
	}
	immediate = {
		if = {
			limit = {
				exists = c:USA

				}
			}

				save_scope_as = foreign_investor
			}
		}

			random_country = {
				limit = {
					country_rank >= rank_value:great_power

			}
		}

		name = the_future_of_china.7.a

	type = country_event
	title = the_future_of_china.100.t
	desc = the_future_of_china.100.d
	flavor = the_future_of_china.100.f
	event_image = {
		trigger = {
			var:china_winner = 1
			var:ztr_civil_war_value = 0
		}
		video = "modernising_china"
	}
	event_image = {
		trigger = {
			var:china_winner = 2
			var:ztr_civil_war_value = 0
		}
		video = "chinese_city"
	}
	event_image = {
		trigger = {
			var:china_winner = 3
			var:ztr_civil_war_value = 0
		}
		video = "asia_buddhism"
	}
	event_image = {
		trigger = {
			var:ztr_civil_war_value = 1
		}
		video = "unspecific_devastation"
	}
	on_created_soundeffect = "event:/SFX/UI/Alerts/event_appear"
	icon = "gfx/interface/icons/event_icons/event_newspaper.dds"
	trigger = {
		has_variable = the_future_of_china_active
	}
	immediate = {
		# Calculate X over Y
		set_variable = {
			name = ztr_civil_war_value
			value = 0
		}
		# 0 means no civil war, 1 means civil war
		# --- CASE: CORPORATE IS X ---
		if = {
			limit = {
				var:ztr_china_corporate_var >= var:ztr_china_federation_var
				var:ztr_china_corporate_var >= var:ztr_china_technoconfucian_var
			}
			set_variable = {
				name = china_winner
				value = 1
			}
			# 1 = Corporate
			every_scope_state = {
				limit = {
					is_coastal = yes
				}
				add_modifier = {
					name = ztr_coastal_construction_bonus_modifier
					years = 10
				}
			}
			if = {
				limit = {
					var:ztr_china_federation_var >= var:ztr_china_technoconfucian_var
				}
				set_variable = {
					name = china_second
					value = 2
				}
				# 2 = Federation
				set_variable = {
					name = diff
					value = var:ztr_china_corporate_var
				}
				change_variable = {
					name = diff
					subtract = var:ztr_china_federation_var
				}
			}
			else = {
				set_variable = {
					name = china_second
					value = 3
				}
				# 3 = Techno
				set_variable = {
					name = diff
					value = var:ztr_china_corporate_var
				}
				change_variable = {
					name = diff
					subtract = var:ztr_china_technoconfucian_var
				}
			}
			if = {
				limit = {
					var:diff <= 20
				}
				set_variable = {
					name = ztr_civil_war_value
					value = 1
				}
			}
		}
		# --- CASE: FEDERATION IS X ---
		else_if = {
			limit = {
				var:ztr_china_federation_var >= var:ztr_china_corporate_var
				var:ztr_china_federation_var >= var:ztr_china_technoconfucian_var
			}
			set_variable = {
				name = china_winner
				value = 2
			}
			# 2 = Federation
